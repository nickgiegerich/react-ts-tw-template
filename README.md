# My Starter Code Template for React, Typescript, and TailwindCSS

### get started with - 

```shell
git clone https://github.com/nickgiegerich/react-ts-tw-template.git
```

### rename directory with - 
```shell
mv path/to/where/file/is/react-ts-tw-template path/to/where/file/is/[your-project-name]
```

### go into the directory
```shell
cd your-project-name
```

### change the name in package.json 

from :

```json
"name": "react-ts-tw-template",
```

 to :
```json
"name": "your-project-name",
```

### run -
```
npm i
```

### make sure it works by running - 
```shell
npm run start
```
